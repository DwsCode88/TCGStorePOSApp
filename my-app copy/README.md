# VaultTrove - TCG Singles Management System

Professional TCG singles intake, pricing, and POS system built with Next.js, Firebase, JustTCG API, and Square.

## Features

- **Singles Intake**: Search and add cards with automatic SKU generation
- **Smart Pricing**: Market-based pricing with customizable rules
- **Label Printing**: Generate professional barcode labels
- **Square POS Integration**: Sync inventory and track sales
- **Dual Modes**: Store and Show modes for different selling contexts
- **Multi-Game Support**: Pok√©mon, MTG, One Piece, Lorcana, and more

## Setup

1. Install dependencies:
```bash
npm install
```

2. Copy environment variables:
```bash
cp .env.example .env.local
```

3. Configure your environment variables in `.env.local`

4. Run development server:
```bash
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000)

See `.env.example` for required configuration.
